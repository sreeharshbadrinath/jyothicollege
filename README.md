# Jyothi Engineering College Showcase

A modern web application showcasing Jyothi Engineering College. This project is built using TypeScript and Vite, with Firebase integrated for backend services and data management.

Based on the repository configuration, this project utilizes the following technologies:

*   **Language:** TypeScript (98.1%), HTML, CSS
*   **Build Tool:** Vite (`vite.config.ts`)
*   **Backend & Database:** Firebase / Firestore (`firebase-applet-config.json`, `firestore.rules`)
*   **Package Manager:** npm (`package.json`, `package-lock.json`)

## 🛠️ Prerequisites

Before you begin, ensure you have the following installed on your local machine:

*   [Node.js](https://nodejs.org/) (v16 or higher recommended)
*   npm (comes with Node.js) or yarn
